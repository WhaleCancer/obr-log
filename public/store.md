# AFF Character Sheet

An Advanced Fighting Fantasy character sheet extension for Owlbear Rodeo, specifically designed for AFF 2nd Edition.

## Features

- **Characteristics**: Track SKILL, STAMINA, LUCK, and MAGIC with Initial/Current values
- **Special Skills**: Organized by category (Combat, Movement, Stealth, Knowledge, Magical)
- **Talents**: Track Major and Minor Talents
- **Drawbacks**: Record Racial Drawbacks
- **Export/Import**: Save and load character data as JSON
- **GM View**: GMs can view all player character sheets

## How to use

- Click on values in the right column to edit them directly
- Click `EDIT` to modify stat names, section titles, and add/remove entries
- Format Characteristics as "Initial/Current" (e.g., "7/7")
- Export your character to JSON and import it back anytime
