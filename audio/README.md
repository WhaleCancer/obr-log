# Shared Audio Repository

Audio files for the AFF extension set live here.

This is the single shared audio repository for:

- `aff-shared-log`
- `star-trek-sheet-main`
- related AFF/Owlbear extensions that use the shared `/audio/...` route

Expected filenames:

- `dice-roll.mp3`
- `tos_keypress2.mp3`
- `tos_keypress7.mp3`
- `damage.mp3`
- `hypospray3_clean.mp3`
- `tos_red_alert.wav`
- `tos-secure-from-red-alert.mp3`

Notes:

- Keep filenames exact.
- The `aff-shared-log` CORS server now serves audio only from this folder.
- If a sound is missing, add the file here rather than in another extension.
